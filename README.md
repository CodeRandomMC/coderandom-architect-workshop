## 🏃‍♂️ Running Locally

```bash
# Install dependencies
pnpm install

# Run development server (with Turbopack)
pnpm dev

# Build for production
pnpm build

# Type checking
pnpm typecheck
```

Open [http://localhost:9002](http://localhost:9002) to view the portfolio.
